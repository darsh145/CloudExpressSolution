Adding 3D T-Shirt Models

This directory is where the 3D T-shirt models should be placed. The application is looking for a file named shirt.glb.

Note: Since we couldn't automatically download the models from the GitHub repository, you'll need to manually add them.

Steps to Add the 3D T-shirt Model:

1. Download the Model

Download the T-shirt model from the GitHub repository: https://github.com/Starklord17/threejs-t-shirt

2. Place the Model Here

Place the downloaded GLB file in this directory (/models/) and rename it to shirt.glb.

3. Restart the Application

Restart the application for the changes to take effect.

Once you've completed these steps, the 3D T-shirt model should load in the 3D view when you press Alt+Q.
